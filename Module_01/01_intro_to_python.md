## 1. 📝 What is Programming?
> **Definition:** *Programming is the process of creating a set of instructions that tell a computer how to perform a task.*

**Explanation (Hinglish):**
Computer apne aap mein smart nahi hota. Use har kaam ke liye "Instructions" chahiye hote hain. Programming wahi language hai jise computer samajhta hai taaki wo hamara kaam (jaise calculations ya data processing) kar sake.

---

## 2. ⚙️ How it Works: Compiler vs Interpreter
Programming language ko machine language (0,1) mein badalne ke liye do main tools hote hain:

### 🔹 Compiler
* **Definition:** *A compiler translates the entire source code into machine code in one go before execution.*
* **Detail:** Ye poore code ko ek saath scan karta hai. Agar kahin error ho, to program run nahi hota. (e.g., C, C++).

### 🔹 Interpreter
* **Definition:** *An interpreter translates and executes the code line-by-line.*
* **Detail:** **Python ek Interpreted language hai.** Ye ek-ek line read karta hai aur execute karta jata hai. Isse debugging (galti dhoondna) bahut fast aur easy ho jata hai.

---

## 3. 🐍 What is Python?
> **Definition:** *Python is a high-level, interpreted, and general-purpose programming language created by Guido van Rossum in 1991.*

**Key Concepts:**
- **High-level:** Iska syntax English jaisa hai, jo humans ke liye readable hai.
- **General-purpose:** Iska use Web, AI, Data Science, aur Automation sab ke liye hota hai.
