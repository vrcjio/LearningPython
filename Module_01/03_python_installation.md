# 03. Python Installation: Anaconda Distribution 🛠️

Professional Data Science aur Python development ke liye **Anaconda** sabse best platform hai.

---

## 1. 📦 What is Anaconda?
> **Definition:** *Anaconda is a distribution of Python and R for scientific computing and data science that simplifies package management and deployment.*

**Why use it?** Isme Python ke saath-saath Jupyter Notebook, Spyder, aur saari important libraries (Pandas, NumPy) pehle se install milti hain.

---

## 2. 📥 Installation Steps

### 🪟 Windows:
1. [Anaconda.com](https://www.anaconda.com/) se Windows installer download karein.
2. Installer run karein aur **"Add Anaconda to my PATH"** option ko tick karein (taki terminal se access ho sake).

### 🍎 Mac:
1. Graphical Installer (.pkg) download karein aur instructions follow karein.

### 🐧 Linux:
1. Terminal mein bash script (.sh) run karein: `bash Anaconda3-xxxx.sh`.

---

## 3. ✅ Verification
Check karne ke liye terminal/CMD mein type karein:
`python --version` aur `conda --version`.
