This is Module 3 
1. Lists (The Mutable Sequence) 📋

List create karna [] aur elements ko access karna (Indexing & Slicing).

List ko modify karna (mutability): append(), insert(), remove(), pop().

Important List methods: sort(), reverse(), count().

Practical application: List ke andar for loop chalana.

2. Tuples (The Immutable Sibling) 🔒

Tuple kya hai aur ise kaise banate hain ().

List aur Tuple ke beech ka sabse bada difference (Immutability - jise badla na ja sake).

Tuple packing aur unpacking (jaise a, b, c = my_tuple).

Tuples ka use kahan aur kyun karna chahiye.

3. Sets (The Unique Collection) 🎯

Set create karna {} aur iska main rule (Duplicate values allow nahi hoti).

Set mein items add aur remove karna.

Mathematical Set Operations: Union (|), Intersection (&), Difference (-), aur Symmetric Difference (^).

4. Dictionaries (Key-Value Pairs) 📖

Key-Value concept ko samajhna (jaise phonebook mein naam aur number).

Dictionary banana {"key": "value"} aur data access karna.

Nayi keys add karna aur purani update karna.

Dictionary methods: .keys(), .values(), aur .items().

Practical application: Dictionary ke keys aur values par ek sath loop chalana.
